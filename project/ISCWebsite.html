Inverness Soccer Website

Introduction

At the begininng of my junior year of high school, the commissioner for the local soccer league asked if I could create a newer looking and easier to use website for the league. I accepted the offer. I worked on the website in the fall of my junior year and towards the end of my junior year of high school.

Development

The website seemed simple, displaying the current information and news for the league, but there was a problem. The commissioner wanted the website to be updated without modifying any HTML code. Since I didn't want to deal with databases or take any security risks, I used PHP to read in text files that would be displayed as HTML. For the styling, I used a Bootstrap CSS top navbar and dropdown templates.
