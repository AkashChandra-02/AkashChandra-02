GMRS Radio Decoding

Introduction

After doing all the usual SDR things, such as hearing radio traffic and picking up satalites, I wanted to do more with the dongle and understand the complexities of radio systems.

Development

I started to look for a python libary. I found pyrtlsdr, which is a python wrapper for the RTL-SDR C libary. I went online and found a tutorial on FM signal processing by Fraida Fund. One thing I noticed is that if I use python, it runs slowly. In order to speed up the program, I decided move to C.

Source Code: The Arduino sketch can be found on GitHub at AkashChandra-02/GMRSRadioDecoding.
