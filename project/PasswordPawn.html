Password Pawn

Introduction

During the 2018 D211 hackathon, I joined with a team of 4. There was a contest, so our team entered. Since we had 2 beginners, we decided to create a password manager that would only use Javascript, HTML5, and CSS. We were given 17 hours to complete to project.

Development

We decided to split the website into two diffrent sections, the "backend" with the encryption and the "frontend" with all the HTML5 and CSS. I chose to work on the backend. I chose to use local storage that was built into chrome. It uses a 32 bit AES key to encrypt the passwords. There weren't any big challenges to over come with the backend, since the libary I was using is easy to use.

Features

  1. 16 character random password generator
  2. Storage of usernames
  3. Local storage of passwords
  4. "Fun" minigame

Source Code: The code can be found on GitHub at AkashChandra-02/PasswordPawn.
