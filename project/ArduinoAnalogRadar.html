Arduino Analog Radar

Introduction

I wanted to design and build something that didn't rely on my computer or the RTL-SDR. Since my friends had radios that worked on the GMRS (General Mobile Radio Service) band, I decided that I would try to display the location of GMRS radios relative to the user with a 12 pixel neopixel ring.

System Map

The antenna will have a spinning cover that will expose a section of the surrounding with an "aperture" to allow the antenna to receive. Using the MOSFET and photointerrupter, I can time how long the "aperture" in the cover is open and correspond it to an LED. The antenna will be tuned to the frequency I want, so in this case I'm going to use one tuned to ~462 MHz. Once the antenna picks up the signal, it is going to be amplified and then I will then pick up it's envelope and send that to the Arduino.

Development

I started with the Arduino sketch, since I was still trying to design the amplifer and envelope detector to feed into the Arduino. The first step I took was to get the Neopixel ring working based off of the given sketches. After I blinded myself with the Neopixel, I started to write the sketch. The functions of the sketch is:

  1. Time how long the motor takes to move the aperture.
  2. Flash the Neopixel green to show that it was ready.
  3. Stop the motor.
  4. In the Loop:
    1. Start the motor spinning again.

Source Code: The Arduino sketch can be found on GitHub at AkashChandra-02/ArduinoAnalogRadar.
